# 🎧 Music Player Manager

A music player manager built using React (Vite) and PocketBase that allows users to upload, manage, and play songs. The app provides a simple dark-themed interface where users can add songs with a title and artist, store them in a database, and play them directly in the browser.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Clone the repository using: git clone <your-repo-url>
2. Install dependencies using: npm install
3. Download PocketBase from: https://pocketbase.io/ and then run: ./pocketbase serve
4. Start the React app using: npm run dev
5. Open http://localhost:5173 in your browser

Instructions for using the software:

1. Open the app in your browser
2. Enter a song title and artist, then select an audio file
3. Click the Upload button to add the song
4. View songs in the playlist
5. Click a song to play it in the audio player

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Node.js (v18 or newer)
* React (with Vite)
* PocketBase (latest version) – https://pocketbase.io/
* Modern web browser (Chrome, Edge, or Firefox)
